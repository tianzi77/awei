#商旅Online原型Git仓库说明文档#
##原型目录说明##
+ __home__： 商旅首页平台
  + __登录页__
  + __登陆后首页__
  + __找回密码__
+ __flight__: 国内机票
+ **flight_int**: 国际机票
+ __hotel__: 国内酒店
+ **hotel_int**: 海外酒店
+ __railway__：火车票
+ __corpadmin__: 商旅用户管理系统
+ __report__: 在线管理报告
  + **date_report_pdf**: 在线管理报告PDF版
+ __mail__: 邮件发送
+ __mytrip__: 我的携程
+ __openui__: 预定页公用模块、交叉推荐公用模块(酒店机票火车票)
+ __ME__: 自助管理平台
+ __car__: 用车
+ __CMS__: 文章管理
+ __lib__: js框架
+ __pack__: 零散页面 
+ ___css__： 公共及组件CSS文件
+ ___src__： 公共及组件SASS源文件
+ ___include__： 公共及组件PHP源文件
+ __docs__： 商旅UI库
